# nowplaying-fields

My personal script that allows me to have a special profile field that updates while I'm playing music.

## Dependencies

* Python 3
* python-decouple
* playerctl
* curl

## Configuration

Here's an example config:

```
TOKEN="YOUR_SUPERSECRET_TOKEN"
INSTANCE="mastodon.social"
FIELDID="3"
NAME="Now playing"
PLAYER="spotify"
```
